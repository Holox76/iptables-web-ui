Ce code est une interface graphique pour iptables, le pare feu Linux.

Elle vous permet de :

* Bloquer une adresse IP
* Autoriser une adresse IP
* Autoriser le ping
* Bloquer le ping
* Bloquer un port
* Autoriser un port
* Autoriser l'UDP
* Bloquer l'UDP
* Vider les règles
* Bloquer le trafic extérieur
